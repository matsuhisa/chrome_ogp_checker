# chrome ogp checker

Chrome の拡張機能。サイトに設定されているmetaタグなどをチェックして表示します。

# インストール

Chrome ウェブストア でインストールできます

* https://chrome.google.com/webstore/detail/ogpchecker/keiplbgeilckhgmkaoodclmcahkkmepm?hl=ja

# チェックできる項目

* title
* h1
* description
* keywords
* og:image
* og:url
* og:image
* og:description
* canonical
